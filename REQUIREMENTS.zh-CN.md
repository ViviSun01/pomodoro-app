# 番茄钟（Pomodoro）需求说明（v1）

## 目标与范围
- **目标**：提供一个视觉统一的扁平插画风番茄钟，支持「专注 / 短休息 / 长休息」三种模式，且用户可在界面内修改各模式时长并保存。
- **范围**：本需求聚焦于核心计时、模式切换、时长配置、基础可用性与 UI/组件风格一致性；不强制包含账号体系、联网同步、统计报表等扩展功能。

## 术语
- **专注（Focus）**：倒计时进行专注。
- **短休息（Short Break）**：每次专注后的小休息倒计时。
- **长休息（Long Break）**：完成若干次专注后的长休息倒计时。
- **一轮（Cycle）**：一次「专注」结束后进入休息的完整过程（休息类型取决于当前轮次规则）。

## 产品体验与视觉风格（扁平插画风）
### 视觉方向
- **风格关键词**：扁平、插画感、柔和渐变/轻噪点、少阴影、圆角、友好。
- **主视觉**：以番茄插画为核心（可参考 `22ef82adc076510e1dc2aaa7c79aa049.jpg` 的简洁形状与配色倾向），用于首页/计时页的背景或中心图形。
- **层级**：用大小、间距、颜色对比构建层级；避免拟物与复杂高光。

### 色彩与排版（建议约束，便于统一）
- **色彩**：
  - 主色：番茄红（用于关键动作、当前模式强调）
  - 辅色：叶子绿（用于图标/强调点缀）
  - 背景：暖黄/米黄（大面积背景，保证舒适）
  - 中性色：用于文本与分割线（保证对比度与可读性）
- **字体**：
  - 数字计时：等宽或数字对齐友好的字体（提升读秒稳定感）
  - 正文：系统字体优先，保证跨平台一致性与渲染清晰
- **圆角与阴影**：
  - 组件圆角统一（例如 12px 或 16px）
  - 阴影极轻或不用阴影；需要分层时优先用浅色背景块与描边

## UI 信息架构（页面/区域）
### 计时主界面（核心）
- **模式切换区**：三段切换（专注/短休/长休），当前模式高亮。
- **计时显示区**：大号倒计时（MM:SS）。
- **主操作区**：
  - 主按钮：开始 / 暂停（同一位置切换状态）
  - 次按钮：重置（回到该模式初始时长）
  - 可选：跳过（直接进入下一阶段：专注→休息 或 休息→专注）
- **状态提示**：如“第 2 次专注 / 下一个：长休息”等（可选但建议）。

### 设置（可并入主界面弹层/抽屉）
允许用户修改并保存以下配置：
- **专注时长**（分钟）
- **短休息时长**（分钟）
- **长休息时长**（分钟）
- **长休息触发规则**：每完成 N 次专注进入一次长休息（默认 4）
- **自动开始**（可选）：
  - 自动开始休息（专注结束后自动进入休息计时）
  - 自动开始专注（休息结束后自动进入专注计时）

## 功能需求（Must / Should）
### Must：模式与计时
- **三模式**：支持专注/短休息/长休息三种模式，并可手动切换。
- **倒计时**：
  - 开始后每秒递减；暂停后停止递减；继续可恢复。
  - 计时为 0 时触发“阶段结束”事件。
- **阶段结束行为**：
  - 专注结束：根据规则进入短休或长休（默认每 4 次专注进入长休）。
  - 休息结束：进入专注。
  - 进入下一阶段时，计时重置为该阶段的配置时长。
- **重置**：将当前阶段的剩余时间恢复为该阶段配置时长，并暂停。
- **修改时长**：
  - 用户在设置中修改各模式时长后，**保存并持久化**（下次打开仍生效）。
  - 当用户修改“当前正在运行的阶段”的时长时：
    - 若计时未开始：直接按新时长显示。
    - 若计时进行中：提供明确规则（建议：立即应用到“剩余时间”或应用到“下一轮”二选一；默认建议“应用到下一轮”，避免突变）。实现时需固定一种并在 UI 上说明。

### Must：可用性与边界
- **输入校验**：
  - 时长仅允许整数分钟（或允许 0.5 分钟需明确），范围建议 1–180。
  - 非法输入给出即时提示并阻止保存。
- **切换模式时**：
  - 若正在计时：需要二次确认（例如“切换将重置当前计时，是否继续？”）。
- **状态持久化**（最低限）：
  - 持久化设置项（时长、长休触发 N、自动开始开关）。
  - 运行态（是否正在计时、剩余时间）可选：若不做恢复，至少在刷新/重启时合理回到该阶段初始值。

### Should：提醒与动效（提升体验）
- **结束提醒**：阶段结束给出声音/桌面通知（若权限允许）；至少有明显视觉提示。
- **动效**：按钮切换、模式高亮切换、番茄插画轻微呼吸/进度动效（扁平风，不要重阴影与强拟物）。
- **快捷键**（可选）：空格开始/暂停，R 重置。

## 组件规范（保证扁平插画风一致性）
### 组件清单（建议）
- **SegmentedControl**：三模式切换（专注/短休/长休）
- **TimerDisplay**：倒计时展示（等宽数字、对齐稳定）
- **PrimaryButton / SecondaryButton**：主次操作（统一圆角、统一高度）
- **IconButton**：小图标按钮（设置/跳过）
- **SettingsDrawer/Modal**：设置面板（输入框、步进器、保存按钮）
- **NumberStepper**：分钟输入（+ / - 或输入框，带范围限制）
- **Toast/InlineHint**：输入错误与提示

### 组件风格约束（必须遵守）
- **形状**：统一圆角、统一描边/分割线样式。
- **颜色**：主按钮/高亮只使用主色体系；禁用态使用中性色并降低对比。
- **图标**：线性/扁平填充二选一并全局统一；线宽一致。
- **间距**：统一 8pt 栅格（8/16/24/32）。

## 数据与存储
- **本地存储键**（建议命名）：
  - `pomodoro.focusMinutes`
  - `pomodoro.shortBreakMinutes`
  - `pomodoro.longBreakMinutes`
  - `pomodoro.longBreakEvery`
  - `pomodoro.autoStartBreak`
  - `pomodoro.autoStartFocus`

## 验收标准（Acceptance Criteria）
- **模式**：能在三模式间切换，当前模式高亮准确。
- **计时**：开始/暂停/继续/重置逻辑正确，显示稳定无跳字。
- **规则**：按“每 N 次专注一次长休”正确触发长休。
- **配置**：能修改三种时长并保存；刷新/重启后设置仍生效。
- **一致性**：组件样式统一，整体符合扁平插画风，不出现突兀的拟物/重阴影组件。

## 非目标（本版本不做）
- 账号登录与云同步
- 复杂统计（周/月报表）
- 多主题皮肤市场
